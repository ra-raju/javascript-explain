# max, min, sqrt, random
---

1. **Math.max(x,y,z.....)**

 দুই বা ততধিক সংখ্যা  থেকে বড় সংখ্যাটা রিটার্ণ করে 
 
 ```javascript
var num = Math.max(29, 49, 83, 7, 094);
console.log(num); // 94
```

2. **Math.min(x,y,z.....)**

 দুই বা ততধিক সংখ্যা  থেকে ছোটো সংখ্যাটা রিটার্ণ করে 
 
 ```javascript
var num = Math.max(29, 49, 83, 7, 094);
console.log(num); // 7
```

3. **Math.sqrt(x)**  

 একটা সংখ্যার (পজিটিভ) স্কয়ার রুট রিটার্ণ করে           

```javascript
var num1 = 25;
var num2 = 49;

console.log(Math.sqrt(num1)); // 5
console.log(Math.sqrt(num2)); // 7
```

4. **Math.random()**

০ থেকে ১ এর ভিতরে রেন্ডম নাম্বার রিটার্ণ করে 

```javascript
console.log(Math.random()); // 0.29581576650836494
console.log(Math.random()); // 0.6758225147225085
console.log(Math.random()); // 0.6367819633969041
```
